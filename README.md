# git
Git je distribuirani sistem kontrole verzija: praćenje promena u bilo kom skupu datoteka, koji se obično koristi za koordinaciju rada između programera koji zajedno razvijaju izvorni kod tokom razvoja softvera. Njegovi ciljevi uključuju brzinu, integritet podataka i podršku za distribuirane, nelinearne tokove posla (hiljade paralelnih grana koje rade na različitim sistemima).
